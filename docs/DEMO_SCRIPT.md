# Engram — Demo video script (~90s)

Goal: in 90 seconds make a judge believe the one thing that matters — **NPCs that
truly remember you, with that memory stored on 0G and recalled across sessions**
(tournament criterion #1: "0G must do real work"). Show it, don't tell it.

- **Length:** 90s (hard cap; judges skim). Keep narration tight.
- **The money shot:** the page **reload** where the NPC greets you by name. Everything
  builds to that.
- **Record at:** https://engram-bay.vercel.app/ — Turbo network, a funded sponsor wallet
  set in Vercel, an AI key set (real Claude dialogue). Have the 📜 Memory panel ready.
- **Tip:** rehearse one full run first; pre-fund the sponsor; pre-connect the wallet so
  you don't film the WalletConnect modal.

---

## Shot list

### 0:00–0:08 — Hook (title + thesis)
- **On screen:** the title screen ("ENGRAM · Aldenmoor"). Optional caption.
- **Caption/VO:** "In every game, NPCs forget you. Engram's don't — and no one can make
  them forget."

### 0:08–0:22 — Enter the world (first person)
- **Do:** connect wallet (or land already connected) → walk through Aldenmoor in first
  person (WASD + mouse), approach an NPC (e.g. Sable).
- **Caption/VO:** "Your wallet is your identity. Walk up to a villager and talk."

### 0:22–0:42 — A memorable interaction
- **Do:** open dialogue. Type something specific and personal, e.g. **"My name is
  AriiBen, remember it."** Send. Show the NPC respond in character. Pick a choice that
  visibly moves trust.
- **Caption/VO:** "Real AI dialogue — and it reacts to what you do. Trust, debts, mood:
  all tracked."

### 0:42–0:55 — Write it to 0G
- **Do:** click **Leave & save** → the banner shows **✓ Saved to 0G · root … · tx …**.
  Briefly let the root/tx hashes be readable.
- **Caption/VO:** "On leaving, the memory is written to 0G Storage — content-addressed
  and auditable by its root hash."

### 0:55–1:18 — THE PAYOFF: reload → it remembers (criterion #1)
- **Do:** **reload the page** (F5) — show the reload happening. Walk back to the same NPC.
  It greets you **by name** ("AriiBen —…"). Then open **📜 Memory**: show trust (e.g.
  57/100) and the interaction log listing your past actions.
- **Caption/VO:** "Reload. New session. It still remembers — because the memory didn't
  live in a database we control. It lives on 0G."

### 1:18–1:30 — Why it matters + roadmap tease
- **On screen:** quick text card.
- **Caption/VO:** "Persistent, player-owned NPC memory on 0G. No reset, no patch, no
  developer can erase it. Next: a persistent world and AI nature agents — all on 0G."
- **End card:** Engram · engram-bay.vercel.app · github.com/engram-zero/engram

---

## What to emphasize (and avoid)
- **Emphasize:** the reload-and-recall moment; the root/tx hashes (proof it's on 0G);
  "auditable", "can't be erased". That's criterion #1, on screen.
- **Avoid claiming:** "cross-device" (not live — pointer is local), "your wallet pays /
  one signature" (writes are sponsored server-side now), or any feature not shown.
  Tournament rules disqualify misrepresenting functionality.
- **Optional B-roll:** a slow first-person pan over the village + campfire for the intro,
  to look like a game and not an app.

NPC greets by name + Memory panel (10s) → end card (2s).
